🎬 MovieHub - Movie Listing & Trailer Platform

A modern, responsive movie website built using HTML, CSS, and JavaScript.
MovieHub allows users to browse movies, watch trailers, and enjoy a premium UI experience with animations and 3D effects.

🚀 Features

🎥 Watch movie trailers (YouTube integration)

🔍 Search movies instantly

❤️ Add to favorites (heart button)

🌙 Light / Dark mode toggle

📱 Fully responsive design

✨ 3D animations & floating effects

🎬 Modal trailer player

🎨 Glassmorphism UI design

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling, animations, 3D effects

JavaScript (ES6) – Functionality & API handling
